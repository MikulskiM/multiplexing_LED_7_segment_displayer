# multiplexing_LED_7_segment_displayer
Program written in C for ATmega8L in order to learn interrupts in AVR microcontrollers

whole PORTD:
PD0 -> a  ; PD1 -> b  ; PD2 -> c  ; ... ; PD6 -> g  ; PD7 -> dp (dot)
PORTC (only 4 bits):
PC1 -> w1 ; PC2 -> w2 ; PC3 -> w3 ; PC4 -> w4 (anodes)
